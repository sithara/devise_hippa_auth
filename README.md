# DeviseHippaAuth

This gem extends Devise to support Hippa Authentication Policies .

## Installation

Add this line to your application's Gemfile:

    gem 'devise_hippa_auth'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install devise_hippa_auth
